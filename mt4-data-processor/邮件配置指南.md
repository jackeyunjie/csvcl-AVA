📧 邮件发送功能配置指南
============================

✅ 邮件功能已完全集成！您只需要简单配置即可使用。

## 🔧 配置步骤

### 1. 编辑 email_config.ini 文件
```ini
[EMAIL]
smtp_server = smtp.qq.com
smtp_port = 587
username = 您的邮箱@qq.com          # ← 改为您的邮箱
password = 您的授权码               # ← 改为授权码（不是登录密码！）
sender_name = MT4数据处理系统
```

### 2. 获取QQ邮箱授权码（推荐）
1. 登录QQ邮箱网页版
2. 点击"设置" → "账户"
3. 找到"POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务"
4. 开启"IMAP/SMTP服务"
5. 生成授权码（16位字符）
6. 将授权码填入 password 字段

### 3. 其他邮箱配置
- **163邮箱**: smtp.163.com, 端口465
- **Gmail**: smtp.gmail.com, 端口587
- **企业邮箱**: 请咨询您的IT部门

## ⚠️ 常见问题

### 问题1: "Connection unexpectedly closed"
**解决方案**: 
- 检查授权码是否正确
- 确认SMTP服务已开启
- 检查网络连接

### 问题2: "Authentication failed"
**解决方案**:
- 使用授权码，不是登录密码
- 检查用户名格式（完整邮箱地址）

### 问题3: 收不到邮件
**解决方案**:
- 检查垃圾邮件文件夹
- 确认收件人邮箱地址正确
- 检查邮箱配额
