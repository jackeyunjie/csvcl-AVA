# CSV文件颜色标记程序使用说明

## 功能描述
此程序可以：
1. 从指定文件夹查找包含特定字符串的CSV文件
2. 筛选创建时间在指定分钟数内的文件
3. **专门针对E2:G40区域**进行颜色标记并导出为Excel文件
4. **自动调整B列宽度**为其他列的1.5倍，提升可读性

## 颜色规则
- **红色背景 + 黑色字体**: 数值为 2, 3, 4, 5, 6, 7
- **淡红色背景 + 黑色字体**: 数值为 10, 11, 12, 13, 14, 15
- **绿色背景 + 黑色字体**: 数值为 -2, -3, -4, -5, -6, -7
- **淡绿色背景 + 黑色字体**: 数值为 -10, -11, -12, -13, -14, -15
- **黄色背景 + 黑色字体**: 数值为 8

## 使用方法

```bash

```

- 时间限制: 5分钟

### 方法2：自定义参数
```bash
python csv_color_marker.py --path "你的文件夹路径" --string "目标字符串" --minutes 10
```

### 参数说明
- `--path`: 搜索的文件夹路径（可选）
- `--string`: 文件名中需要包含的字符串（默认: KVBt_@_D1）
- `--minutes`: 时间限制（分钟，默认: 5）

## 输出文件
程序会在当前目录下生成以 `_colored.xlsx` 结尾的Excel文件，包含原始数据和颜色标记。

**特别说明**：
- 程序只会对 **E2:G40 区域** 内的单元格进行颜色标记，其他区域不受影响
- **B列宽度**自动调整为其他列的1.5倍（B列: 18, 其他列: 12）

## 示例
如果找到文件 `KVBt_@_D1_data.csv`，程序会生成 `KVBt_@_D1_data_colored.xlsx`。

程序会扫描E2:G40区域内的所有单元格，并按照颜色规则进行标记。例如：
- E3单元格如果数值为2，将被标记为红色
- F5单元格如果数值为-4，将被标记为绿色
- G10单元格如果数值为8，将被标记为黄色
- E8单元格如果数值为12，将被标记为淡红色
- F12单元格如果数值为-13，将被标记为淡绿色
