## Boundary

- This package contains no raw market data.
- This package contains no DuckDB, parquet, CSV, zip source data, account data, API key, token, or broker credential.
- `state_hex` is a component encoding, not a linear strength score and not a standalone action permission.
- D1, H1, M15, and lower periods are independent viewpoint Agents. A lower-period Agent must build its own strict-prior higher-timeframe context using its own timestamp and close as the viewpoint base for `position`, while `base/trend/volatility` still come from each structure timeframe.
