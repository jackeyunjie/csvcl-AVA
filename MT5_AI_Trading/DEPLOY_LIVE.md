## 部署步骤

### 1. 编译 EA

1. 打开 MT5 → 导航器 → MQL5 → Experts
2. 找到 `AI_Trading_Bridge.mq5`
3. 双击打开，按 **F7** 编译
4. 确认编译成功（0 errors, 0 warnings）

### 2. 加载 EA

1. 图表窗口 → 右键 → 交易品种 → 搜索 US_500
2. 打开 US_500 H1 图表
3. 导航器 → Experts → AI_Trading_Bridge → 双击拖拽到图表
4. 在弹窗中勾选：
   - [x] 允许实时自动交易
   - [x] 允许导入DLL
5. 点击 **确定**

### 3. 确认 EA 运行

图表右上角应显示：**AI_Trading Bridge [5565/5566]**

### 4. 启动 Python 实盘引擎

```bash
cd "d:\qoder\csvcl - AVA\MT5_AI_Trading"
python run_live.py --live
```

或使用一键启动：
```bash
start_live.bat
# 选择 2. 实盘模式
```

1. **Python 端**: 按 `Ctrl+C`
2. **MT5 端**: 图表上右键 EA → 删除

## 首次实盘建议

1. **先用模拟账户测试** 1-2 天
2. **小仓位** 开始（0.01手）
3. **监控 12小时平仓** 是否正常
4. 确认信号与预期一致后再加大仓位
