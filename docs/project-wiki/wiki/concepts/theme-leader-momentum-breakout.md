# 概念：主题龙头动量突破

## 一句话定义

**主题龙头动量突破** 是一种短线交易模式：在市场出现明确主导主题时，只买入该主题中涨幅最大、关注度最高的一等龙头标的，当其以显著放量的长阳线突破六个月高点或历史高点后，在随后两个交易日内追涨或回调买入，并严格执行短线止盈止损纪律。

## 核心逻辑

### 1. 主题效应

- 股市和币市中，资金总是围绕某些主题/板块轮动。
- 即使大盘下跌，市场主导主题也可能逆势走强。
- 主题的形成可以通过 **成交额排名 + 涨幅联动** 来确认。

### 2. 龙头溢价

- 同一主题内，涨幅最大的龙头通常享有：
  - 更高的关注度
  - 更强的资金流入
  - 更足的上涨动量
  - 更好的回调承接
- 跟风标的虽然看似“便宜”，但弹性差、资金回流时容易大跌。

### 3. 新高突破

- 突破六个月高点或历史高点意味着上方无套牢盘抛压。
- 大量突破资金和追涨资金会在突破时形成共识买盘。
- 巨量长阳线是资金抢筹的强烈信号。

### 4. 短线窗口

- 突破后的第 1-2 个交易日是最佳交易窗口。
- 第 3 个交易日起热度衰减，需严格执行出场纪律。

## 关键要素

| 要素 | 要求 | 说明 |
|------|------|------|
| 主题确认 | 前 15 成交额中多只同主题标的涨幅 > 5% | 可扩展至前 30，但不可勉强 |
| 龙头选择 | 主题内涨幅最大、走势最强 | 只交易一等龙头 |
| 突破形态 | 长阳线突破六个月高点或历史高点 | 涨幅 > 10% 更佳 |
| 成交量 | 显著放量 | 至少为近期均量 1.5 倍以上 |
| 入场时机 | 突破后 1-2 个交易日 | 回调支撑或再次突破 |
| 出场纪律 | 3% 止损、5% 止盈、破开盘 1% 警惕 | 短线不贪婪 |

## 常见错误

- 在主题不明确时勉强交易。
- 追买二等、三等跟风标的。
- 在一等龙头涨停时转向二等股。
- 突破无量或假突破时入场。
- 盈利后不愿离场，把短线做成长线。
