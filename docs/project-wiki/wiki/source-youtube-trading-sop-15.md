# 来源：YouTube 交易 SOP《Hong Inki 主题龙头动量突破策略》

## 一句话定义

YouTube 视频 [TQTBv-4fN-k](https://www.youtube.com/watch?v=TQTBv-4fN-k) 拆解了韩国股币双栖交易员 **Hong Inki** 的短线策略：通过成交量排名识别当日市场主导主题，只交易主题内涨幅最大的一等龙头，在其首次以巨量长阳线突破前高时入场，并在后续两个交易日内执行严格的短线出场纪律。

## 为什么保留

该策略把 **主题识别、龙头筛选、突破确认、短线持仓管理** 整合为一套可量化的规则，与前面基于技术指标的 SOP 形成互补。它强调“只追最强标的、只在突破新高时进场、只在两天窗口内交易”，非常适合动量型和事件驱动型交易者的知识库补充。

## 关键结论

### 策略哲学

- 短线交易确定性高于长线持有：可随时处于最佳状态，规避不可预见的长期风险。
- 核心任务：找到当日所有人都在关注的 **市场主导主题** 及其 **一等龙头标的**。
- 牛熊皆可获利：只要存在市场主导主题，即使指数下跌，相关龙头也可能逆势大涨。

### 主题识别规则

1. 观察当日 **交易额排名前 15** 的标的。
2. 筛选其中 **涨幅超过 5%** 的标的。
3. 如果前 15 中有多只标的属于 **同一主题/板块**，即确认当日主导主题。
4. 若前 15 无法确认，可扩大范围；但 **没有符合条件时一定不能勉强**。

### 龙头筛选规则

- 只交易主题内 **涨幅最大的一等标的**。
- 涨幅小的二等标的看似便宜，实际走势缓慢、弹性差。
- 一等标的动量更足、关注度更高，回调时也有更多买盘承接，抗跌性更强。
- 当一等股涨停时，不要去追二等股；一旦一等股开板，资金会回流一等股，二等股容易大跌。

### 最佳入场模型：巨量长阳线突破新高

- 形态：一等标的 **首次突破历史高点/六个月高点** 的 **长阳线**。
- 量能：伴随 **显著放量**。
- 幅度：涨幅 **超过 10%** 更佳。
- 逻辑：突破新高 + 巨量长阳 = 大量突破资金 + 共识买盘聚集 + 追涨资金推动。
- 入场方式：
  - 长阳线突破后，在随后两个交易日内寻找机会。
  - 回调至支撑位入场；或
  - 价格再次突破前高时入场。

### 出场纪律（长阳线后第 3 个交易日开始执行）

| 情形 | 操作 |
|------|------|
| 出现十字星、倒锤子线或下跌 | 价格跌破开盘价 1% 考虑出场 |
| 下跌 3% | 一定出场 |
| 上涨 5% 以上 | 一定离场（短线不贪婪） |

> 核心心态：时刻谨记自己是一位高贵的短线交易员，见好就收。

### 案例要点

- 机器人主题案例（2023-08-31）：彩虹机器人是一等龙头，次日最高涨 12.74%；而同主题二等标 Posco DX、Neuromeca 次日收跌。
- Isupetesis 案例（2023-05-25）：突破历史新高、单日涨 17.1%，两个月内上涨 236%。
