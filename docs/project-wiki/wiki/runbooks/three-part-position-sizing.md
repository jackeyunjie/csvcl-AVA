# 仓位管理 SOP：三分法

## 一句话定义

将交易资金平均分成三份，每次只用其中一份开仓；亏损后补足，盈利后提走，始终保持每份仓位金额固定，作为强制风险隔离和心态稳定器。

## 适用场景

- 高波动市场（加密货币、期货）
- 胜率不高但盈亏比优秀的策略
- 需要控制极端行情回撤的交易员

## 操作规则

### 资金分割

假设总资金为 30,000 USDT：

```text
总资金 = 30,000 USDT
每份资金 = 10,000 USDT
```

### 开仓规则

- 每次开仓只用 **1 份资金**（10,000 USDT）
- 不要三份全部同时用于一笔交易

### 盈亏处理

| 情况 | 操作 |
|------|------|
| 亏损 1,000 | 补入 1,000，保持该份为 10,000 |
| 盈利 1,000 | 提走 1,000，保持该份为 10,000 |

### 加仓规则

- 只有当总资金从 30,000 增长到 60,000 时
- 才可以将每份仓位提高到 15,000 或 20,000
- 加仓前必须确认心理能承受更大波动

## 核心优势

### 1. 强制第二层保险

- 即使遇到插针或极端行情
- 最多只爆掉三分之一的资金
- 避免单笔交易毁灭整个账户

### 2. 心态稳定

- 固定仓位让亏损和盈利时都能保持平和
- 不会因为一次大亏而情绪失控
- 也不会因为一次大赚而过度自信

### 3. 容错率高

- 胜率只有 42% 也能长期盈利
- 关键在于分仓后的盈亏比和连续性

## 与其他风控手段的配合

| 风控手段 | 作用 |
|----------|------|
| 三分法仓位 | 控制单笔最大损失 |
| 止损 | 限制单笔亏损幅度 |
| 低倍杠杆 | 降低爆仓概率 |
| 不逆势交易 | 避免方向性错误 |

## 注意事项

- 加仓必须基于账户增长，而不是情绪冲动
- 提高仓位后要确保心脏能承受波动
- 即使使用三分法，每笔交易仍需设置止损
- 该方法不是鼓励频繁交易，而是提高容错率
