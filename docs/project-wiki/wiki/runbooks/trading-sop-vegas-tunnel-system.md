# 交易 SOP：Vegas 隧道交易系统

## 适用场景

- 品种：BTC/USD、ETH/USD 等高波动趋势型品种；理论上可推广至外汇、股指、商品期货。
- 周期：15M、1H、日线均可；小周期止损更敏感，大周期盈亏比更高。
- 风格：趋势跟踪 / 均线回归 / 回调入场。

## 前置准备

### 数据需求

| 数据 | 说明 |
|------|------|
| K 线 OHLCV | 目标周期的完整历史与实时数据 |
| 成交量数据 | 用于计算 Volume Oscillator |

### 工具设置

- TradingView / MT5 / MT4 / Python 回测平台
- 添加以下指标：
  - EMA 12（黄色，过滤线）
  - EMA 144 / 169（白色，Vegas 隧道）
  - EMA 576 / 676（红色，大趋势线）
  - Volume Oscillator（短线=1，长线=14，零线=144；<1H 可设为 89）

## 操作流程

### 步骤 1：判断趋势方向

1. 观察 EMA 144/169（Vegas 隧道）与 EMA 576/676 的相对位置。
2. **多头排列**（隧道在 576/676 之上）→ 只考虑做多。
3. **空头排列**（隧道在 576/676 之下）→ 只考虑做空。
4. 若两组均线缠绕、方向模糊 → 空仓或减少交易。

### 步骤 2：确认过滤线位置

1. 多头排列时，EMA12 必须在 Vegas 隧道上方。
2. 空头排列时，EMA12 必须在 Vegas 隧道下方。
3. 若 EMA12 与隧道方向不一致 → 放弃操作。

### 步骤 3：等待价格回踩入场

1. 多头：价格回落至 Vegas 隧道附近 → 观察止跌信号后入场。
2. 空头：价格反弹至 Vegas 隧道附近 → 观察止涨信号后入场。
3. 可设置价格区间分批挂单，避免踏空。

### 入场条件

#### 多头入场

1. EMA 144/169 在 EMA 576/676 之上（多头排列）。
2. EMA12 在 Vegas 隧道上方。
3. 价格回踩 Vegas 隧道附近。

#### 空头入场

1. EMA 144/169 在 EMA 576/676 之下（空头排列）。
2. EMA12 在 Vegas 隧道下方。
3. 价格反弹至 Vegas 隧道附近。

### 止损

**价格跌破隧道后的处理流程（以多头为例，空头相反）：**

1. **第一步：Volume Osc 确认**
   - 若 Volume Oscillator 突破 144 零线 → 立即止损。
   - 若未突破 → 进入第二步。
2. **第二步：24 小时加速观察**
   - 若 24 小时内出现加速下跌 → 止损。
   - 若无明显加速 → 持仓，止损移至短期低点/支撑位。
3. **第三步：时间止损**
   - 若 24 小时内价格无法回到 Vegas 隧道上方 → 止损。
4. **最终硬止损**
   - 小周期：可设在 EMA 676 下方。
   - 任何周期：价格跌破 EMA 676 → 必须止损。

### 止盈

- 分批止盈：风险回报比达到 1:1 时平掉部分仓位。
- 剩余仓位持有，博取趋势延续的更大利润。
- 可在趋势运行中根据回踩信号再次加仓，但单次趋势中加仓不超过 2 次。

## 仓位与风险管理

- 单笔风险建议不超过账户净值的 1%–2%。
- 使用最终止损位计算仓位规模：
  `仓位 = 可承受亏损金额 / (入场价 - 止损价)`
- 趋势不明显或均线缠绕时，降低仓位或空仓。

## 回测与优化要点

- 重点测试不同周期（15M / 1H / D1）下的假突破过滤效果。
- 优化 Volume Oscillator 零线参数：144 vs 89。
- 对比“立即止损”与“24 小时观察后再止损”的盈亏比差异。
- 统计均线缠绕区间的亏损次数，用于设计空仓过滤器。

## 注意事项

- 该系统在强趋势行情中表现最佳，震荡行情会导致连续小幅亏损。
- 小周期信号更频繁，但假突破更多，需严格执行 Volume Osc 确认。
- 任何周期价格跌破 EMA676 都是必须遵守的硬止损纪律。
