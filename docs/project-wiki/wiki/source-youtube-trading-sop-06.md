# 来源：YouTube 交易 SOP《Umar Ashraf 日内交易五步法》

## 一句话定义

YouTube 视频 [Z7BRl3W_DHc](https://www.youtube.com/watch?v=Z7BRl3W_DHc) 拆解了日内交易员 **Umar Ashraf** 的五个固定交易流程步骤，核心强调：切实可行的目标、交易前宣誓、清晰计划与执行、固定风险、持续复盘。

## 为什么保留

前面几个视频主要讲指标和入场规则，这个视频补上**交易流程、心理建设和风险管理**维度。一个完整的交易系统不仅需要信号，还需要流程纪律。这个内容与本项目的“邮件报告、定时任务、数据处理流程”有呼应价值。

## 关键结论

### Umar Ashraf 交易五步法

| 步骤 | 内容 | 核心要点 |
|------|------|----------|
| 1. 交易前准备 | 设定现实目标 + 建立晨间例行程序 | 避免不切实际的目标影响决策 |
| 2. 交易宣誓 | 自我承诺，强化执行 | 提醒全天不偏离交易计划 |
| 3. 交易计划与执行 | 不预测，只根据价格行为反应 | 提前标注大级别关键位，计划进/止/盈 |
| 4. 仓位与风险范围 | 固定每笔风险额度 | 选择不会让情绪波动的风险金额 |
| 5. 记录与复盘 | 持续优化交易方式 | 长期盈利来自不断复盘改进 |

### 核心理念

- 心理因素可能占长期盈利的 90%
- 不预测行情，只在关键区域（支撑/阻力/供需区）做突破或波段
- 只吃确定性高的一段走势，不贪婪
- 固定风险额度，保持交易状态稳定
- 交易是长期过程，前期盈利可能只占总盈利的 10%-20%

### 实战案例

- 标的：亚马逊 1 分钟图
- 价格从 127 反弹至 130 附近大周期供应区，未放量突破
- 在供应区设空单
- 止损：130.5
- 止盈：128.8（下方顶底转换结构位）
