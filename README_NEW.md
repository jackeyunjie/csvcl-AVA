# MT4数据自动处理系统

## 🎯 项目简介

这是一个专业的MT4交易数据自动处理系统，专为金融数据分析师和自动化交易系统用户设计。

### 🔥 核心功能
- 📊 **智能数据筛选**: 自动查找包含特定字符的CSV文件
- 🎨 **颜色标记**: 对E2:G40区域进行智能颜色标记
- 📸 **高清截图**: 生成A1:M40范围的完整表格截图
- 📧 **自动邮件**: 7:00-22:00时段自动发送处理结果
- 📈 **数据分析**: 智能识别重要数值变化(2,6,-2,-6)

## 🚀 快速开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 配置邮箱
```bash
# 复制配置模板
cp email_config.ini.template email_config.ini

# 编辑配置文件，填入您的邮箱信息
# 注意：如果使用QQ邮箱，password需要填写授权码
```

### 3. 运行系统
```bash
# 自动模式（推荐）
python auto_email_config.py

# 手动处理模式
python process_real_mt4_data.py
```

## 📋 功能详情

### 颜色标记规则
- 🔴 **红色背景**: 数值为 2, 3, 4, 5, 6, 7
- 🌸 **淡红色背景**: 数值为 10, 11, 12, 13, 14, 15
- 🟢 **绿色背景**: 数值为 -2, -3, -4, -5, -6, -7
- 🍃 **淡绿色背景**: 数值为 -10, -11, -12, -13, -14, -15
- 🟡 **黄色背景**: 数值为 8
- 📐 **处理区域**: E2:G40

### 数据变化分析
- 📊 每三行为一个品种（今天、昨天、前天）
- 🔍 监控重要数值变化：2, 3, 4, 5, 6, 7, 10-15, -2, -3, -4, -5, -6, -7, -10至-15
- 📝 自动生成变化报告并在邮件中突出显示

### 邮件功能
- ⏰ **发送时间**: 7:00-22:00
- 📎 **附件内容**: Excel文件（含颜色标记）
- 🖼️ **嵌入图片**: 高清截图直接显示在邮件正文
- 👥 **多收件人**: 支持同时发送给多个邮箱

## ⚙️ 配置说明

### 文件筛选配置
- **文件特征**: 包含 `KVBt_@_D1` 字符
- **时间限制**: 10分钟内创建的文件
- **文件格式**: CSV格式

### 图片生成配置
- **尺寸**: 1080×1025像素
- **格式**: JPG（质量95%）
- **范围**: A1:M40完整表格
- **特色**: 包含行号列号边框

## 🛡️ 安全注意事项

1. **配置文件安全**：
   - `email_config.ini` 包含敏感信息，不会上传到代码仓库
   - 请妥善保管您的邮箱授权码

2. **数据隐私**：
   - 生成的Excel和图片文件不会提交到版本控制
   - 建议定期清理临时文件

## 🔧 技术栈

- **Python 3.x**: 核心开发语言
- **pandas ≥1.3.0**: 数据处理
- **openpyxl ≥3.0.0**: Excel操作
- **Pillow ≥8.0.0**: 图像处理
- **smtplib**: 邮件发送
